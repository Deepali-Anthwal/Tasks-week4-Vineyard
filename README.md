# Interactive Q&A Webpage - Week 4 Task of Vineyard Program

## Project Description
It focuses on Object-Oriented Programming (OOP) in JavaScript to create a modular and scalable FAQ interface. By encapsulating data and logic within a class, the application demonstrates efficient DOM manipulation and state management while providing FAQs on Data Structures.

## Implementation Details
* **HTML5:** Used semantic elements for structure and unique ID attributes for efficient JavaScript DOM targeting.
* **OOP Logic:** Developed QAItem class to encapsulate question/answer data and visibility states.
* **CSS3:** Focused on a functional, high-contrast design to ensure readability and clarity with class-based toggles.
* **JavaScript (ES6):** Leveraged Class constructors, template literals for dynamic rendering, and arrow functions.

## Interactivity Features
* **State Management:** Uses an internal isOpen boolean to track the toggle state of each individual question.
* **Dynamic Content:** Renders a dataset of  DSA FAQs directly into the DOM using class methods.
* **Visual Cues:** Features a dynamic icon system (+/−) that updates in real-time based on the expansion state.

## File Structure
* **index.html:** Main HTML document containing the UI structure and the display container.
* **style.css:** Stylesheet for the clean layout, rounded containers, and high-contrast typography.
* **script.js:** Core logic file containing the `QAItem` class and the DSA dataset.

## How to Run
Download the source files. Make sure all the 3 files, index.html, script.js and style.css are in the same folder. Launch index.html in any browser. You will be able to see the Interactive Q&A Wepage.
